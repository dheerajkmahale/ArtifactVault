# ArtifactVault - 3D Artifact Scanning & Digital Preservation

An intelligent system for 3D artifact scanning and digital preservation using AI-powered classification and visualization.

## About

ArtifactVault transforms the way museums, archaeologists, and cultural institutions preserve and share their artifacts through:

- **2D to 3D Conversion**: Advanced AI transforms 2D images into interactive 3D models
- **AI Classification**: Automated categorization and analysis of artifacts
- **Digital Preservation**: Secure storage and management of cultural heritage
- **Interactive 3D Viewer**: Web-based 3D visualization and exploration

## Technology Stack

- **Frontend**: React 18 + TypeScript + Vite
- **UI Components**: shadcn/ui + Tailwind CSS + Radix UI
- **3D Graphics**: Three.js for web-based 3D rendering
- **Backend**: Supabase (Database, Authentication, Storage, Edge Functions)
- **State Management**: TanStack React Query
- **Forms**: React Hook Form + Zod validation

## Quick Start

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Modern web browser

### Installation & Development

```bash
# Clone the repository
git clone <repository-url>
cd artifact-3d-studio-main

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your Supabase credentials

# Start development server
npm run dev
```

The application will be available at `http://localhost:8080`

### Environment Setup

Create a `.env` file with your Supabase credentials:

```env
VITE_SUPABASE_PROJECT_ID=your-project-id
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=your-anon-key
```

### Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Create production build
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint code quality checks

## Supabase Setup

### Required Tables
- `artifacts` - Store artifact metadata and processing status
- `auth.users` - User authentication (auto-generated by Supabase)

### Required Storage
- `artifacts` bucket for uploaded images and 3D models

### Required Edge Functions
- `classify-artifact` - AI classification processing
